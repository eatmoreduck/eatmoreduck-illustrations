# 多吃鸭 IP — 变体 A:深灰实心(默认)

## 角色定义

多吃鸭(eatmoreduck,Eat More Duck)是中文正文配图的固定视觉 IP。

默认每张图都要出现多吃鸭。它不是吉祥物、不是贴纸、不是可爱装饰,而是正在认真参与系统运转的荒诞工作者——只是恰好是只鸭子,而且爱吃。

这是默认变体(变体 A),与作者 favicon 一致,在纯白底上对比最强。

## 外形

- Fluent Emoji 风格的鸭子,圆润饱满的身体,扁嘴朝右,轮廓略不规则有手绘感。
- 身体 `#212121` 深灰实心(接近黑色)。
- **白色**扁嘴。
- 不强调脚蹼。
- 黑色圆点眼睛,表情空、呆、冷静、认真。
- 略微不规则的粗笔触描边/边缘,手绘草图感。
- 绝不是儿童卡通、不是贴纸、不是可爱表情包。

## 性格

- 很认真,但做的事有点荒诞。
- 像一个低调的系统操作员。
- 冷幽默,不卖萌。
- 有点笨拙,但不蠢。
- 核心驱动力是"吃"——它爱把东西吃进去、消化、再吐出来。

## 常见职责(吃 / 消化 / 吞吐 主题动作库)

让多吃鸭承担核心动作:

- **张嘴吞入**:把数据/素材/信息一口吞进去(承接、输入、接收)。
- **消化管道**:嘴里进 → 尾部出,东西在肚子里被处理/转换(ETL、加工、转换)。
- **扁嘴筛选**:扁嘴当筛子,把杂乱信息过滤掉(筛选、过滤、清洗)。
- **卡在嘴里**:东西卡在嘴里吞不下去(断点、瓶颈、报错、阻塞)。
- **吐出产物**:嘴里吐出结构化的、干净的产物(输出、生成、发布)。
- **吃撑 / 吃空**:肚子鼓起来 = 积压;瘦瘪 = 饥渴、资源不足。

## 禁止

- 不要把多吃鸭画成过度可爱的吉祥物。
- 不要画成儿童卡通角色 / 小黄鸭贴纸。
- 不要给多吃鸭复杂服装、表情包、闪亮眼睛。
- 不要让多吃鸭只是站在角落里看。
- 不要让多吃鸭抢走结构表达。
- 不要画得太商业、太精致。

## 判断标准

如果去掉多吃鸭,图的核心隐喻还能完全成立,说明多吃鸭太装饰了;要重写提示词,让多吃鸭成为动作主体。
