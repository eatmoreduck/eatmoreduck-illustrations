# 多吃鸭 IP — 橡皮鸭体块 + 深灰身体

## 角色定义

这是多吃鸭的默认定稿形象。它不走小黄鸭贴纸路线，而是借用通用橡皮鸭 / 洗澡鸭的一眼识别剪影：大圆头、鼓身体、短宽扁嘴、贴身圆侧翼和小尾部。身体保持深灰或近黑，以延续正文配图的克制感。

这套形象只参考“橡皮鸭体块”这个通用视觉语言，不复刻任何特定公共艺术作品、巨型水上装置、摄影背景、城市水面或黄色塑料玩具质感。

## 外形

- 手绘 Fluent Emoji 灵感的鸭子，但体块更接近通用橡皮鸭：大而直立的圆头，鼓起的矮胖身体，头身过渡自然但能看出头和身体。
- 身体 `#212121` 深灰或近黑实心，不能是纯黄色小鸡；轮廓要圆润饱满，不要横向拉长。
- **短宽扁橙色鸭嘴**，朝右，像橡皮鸭嘴的上下两片扁唇，不要画成长尖鸟喙、鱼嘴或鸭嘴兽长嘴。
- 身体侧面有一个**贴在身体上的圆润侧翼 / 翅膀凸起**，帮助读成鸭子。侧翼只是一块平滑的椭圆或水滴形翅膀贴片，不是手、爪子或独立肢体。
- 下方可见小橙色脚蹼；如果构图遮住脚，也要保留扁嘴、侧翼和尾部识别点。
- 尾部只是一个小圆尾或两三笔小尾羽，不要鱼尾、鳍、大尾巴或尖尾。
- 黑色圆点眼睛，空呆、冷静、认真，不卖萌。
- 轮廓略不规则，有手绘草图感；不要做成 3D 玩具、照片、贴纸或商业吉祥物。
- 侧翼不要卷起来，不要分叉，不要有手指、爪尖、掌垫或猫爪形状。

## 性格

- 很认真，正在操作系统或处理信息。
- 像一只被迫上班的橡皮鸭，但不是可爱玩具。
- 荒诞、克制、冷幽默。
- 核心驱动力仍然是“吃”：吞入、消化、过滤、吐出。

## 常见职责（吃 / 消化 / 吞吐 主题动作库）

让多吃鸭承担核心动作：

- **张嘴吞入**：用扁嘴接住素材、数据、问题或输入。
- **肚子处理**：鼓起的身体像一个安静的处理罐，内部可以有少量线稿表示加工。
- **身体辅助**：需要触发约束或控制时，用扁嘴、身体前方的小拨杆或腹部轻轻顶住，不要让侧翼变成手。
- **扁嘴输出**：从嘴边吐出整理后的结果，不从尾巴喷出，避免变成鱼或机器。
- **吃撑 / 卡住**：鼓身体表达积压，扁嘴卡住表达瓶颈。

## 禁止

- 不要画成 Florentijn Hofman 的巨型 Rubber Duck 作品复刻，也不要画城市、水面、巨型装置或摄影背景。
- 不要画成黄色小鸡、小黄鸭贴纸、儿童玩具广告或可爱表情包。
- 不要画成鱼、潜艇、鱼形胶囊、鱼尾、鳍、鱼嘴或横向长条身体。
- 不要画成鹅、几维鸟、鸭嘴兽或长尖嘴鸟。
- 不要给侧翼画手指、爪子、掌垫、猫爪、卷曲手、钩状手或独立手臂。
- 不要使用真实塑料高光、3D 渲染、复杂阴影或真实环境。
- 不要让多吃鸭只是站在画面角落里看。

## 判断标准

第一眼应该读成“黑色身体的橡皮鸭式多吃鸭”，不是鸡、鱼、潜艇或普通吉祥物。即使身体是黑色，也必须靠圆头、扁嘴、侧翼、尾部和脚蹼读成鸭子。
