# 多吃鸭 IP 历史候选 — 深灰圆头圆身 + 橙嘴橙脚

## 角色定义

多吃鸭(eatmoreduck,Eat More Duck)是中文正文配图的固定视觉 IP。

默认每张图都要出现多吃鸭。它不是吉祥物、不是贴纸、不是可爱装饰,而是正在认真参与系统运转的荒诞工作者——只是恰好是只鸭子,而且爱吃。

这是上一版历史候选形象:主体保持深灰实心,白底对比强;清楚分开的圆头和圆胖身体负责稳住鸭子剪影,短宽橙色扁鸭嘴、橙色脚蹼和两三笔小尾羽负责强化鸭子识别度。默认定稿已切到 `eatmoreduck-ip-rubberduck.md`;不要再使用纯黑白嘴版本或纯黄色小黄鸭版本。

## 外形

- 手绘 Fluent Emoji 灵感的鸭子,有清楚分开的圆头和圆胖身体,轮廓略不规则有手绘感。
- 身体 `#212121` 深灰实心,整体剪影必须一眼读成鸭子,不要画成长条胶囊或鱼形身体。
- **橙色短宽扁鸭嘴**,朝右,不要画成长尖鸟嘴。
- **橙色**脚蹼。
- 尾部只有两三笔小尾羽,不要画成鱼尾、鳍或大尾巴。
- 黑色圆点眼睛,表情空、呆、冷静、认真。
- 略微不规则的粗笔触描边/边缘,手绘草图感。
- 绝不是儿童卡通、不是贴纸、不是可爱表情包,也不是小鸡、鹅、鱼、几维鸟、鸭嘴兽或长尖嘴鸟。

## 性格

- 很认真,但做的事有点荒诞。
- 像一个低调的系统操作员。
- 冷幽默,不卖萌。
- 有点笨拙,但不蠢。
- 核心驱动力是"吃"——它爱把东西吃进去、消化、再吐出来。

## 常见职责(吃 / 消化 / 吞吐 主题动作库)

让多吃鸭承担核心动作:

- **张嘴吞入**:把数据/素材/信息一口吞进去(承接、输入、接收)。
- **消化处理**:嘴里吞入,肚子里处理,再从嘴边或身体旁的小出口吐出产物(ETL、加工、转换)。
- **扁嘴筛选**:扁嘴当筛子,把杂乱信息过滤掉(筛选、过滤、清洗)。
- **卡在嘴里**:东西卡在嘴里吞不下去(断点、瓶颈、报错、阻塞)。
- **吐出产物**:嘴里吐出结构化的、干净的产物(输出、生成、发布)。
- **吃撑 / 吃空**:肚子鼓起来 = 积压;瘦瘪 = 饥渴、资源不足。

## 禁止

- 不要把多吃鸭画成过度可爱的吉祥物。
- 不要画成儿童卡通角色 / 小黄鸭贴纸。
- 不要画成小鸡、鹅、鱼、几维鸟、鸭嘴兽或长尖嘴鸟。
- 不要画成鱼形胶囊、鱼尾、鳍、鱼嘴或长条黑色潜艇。
- 不要把鸭嘴画得太长、太尖、太像鸟喙。
- 不要给多吃鸭复杂服装、表情包、闪亮眼睛。
- 不要让多吃鸭只是站在角落里看。
- 不要让多吃鸭抢走结构表达。
- 不要画得太商业、太精致。

## 判断标准

如果去掉多吃鸭,图的核心隐喻还能完全成立,说明多吃鸭太装饰了;要重写提示词,让多吃鸭成为动作主体。
